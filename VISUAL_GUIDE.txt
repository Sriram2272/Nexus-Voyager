HOW TO TELL IF BACKEND IS RUNNING
═══════════════════════════════════════════════════════════════════════

✅ GOOD SIGNS (Backend is running):
   • See window titled "Nexus AI Backend - Keep This Open"
   • Window shows "Running on http://localhost:5000"
   • Window shows "Press CTRL+C to quit"
   • Frontend in browser responds to questions
   • No error in browser chat

❌ BAD SIGNS (Backend is NOT running):
   • No backend window visible
   • Browser shows "❌ Sorry, I encountered an error"
   • Browser shows "Please make sure the backend server is running"
   • Questions don't get answered


TROUBLESHOOTING VISUAL GUIDE
═══════════════════════════════════════════════════════════════════════

Problem: "Backend server is running" error
────────────────────────────────────────────────────────────────────
    Browser Shows:
    ┌─────────────────────────────────────┐
    │ ❌ Sorry, I encountered an error.   │
    │ Please make sure the backend        │
    │ server is running.                  │
    └─────────────────────────────────────┘
    
    Solution:
    1. Look for backend window (green text)
    2. If missing → Run START_NEXUS_AI.bat again
    3. Wait 10 seconds
    4. Refresh browser


Problem: Nothing happens when I ask a question
────────────────────────────────────────────────────────────────────
    Browser Shows:
    ┌─────────────────────────────────────┐
    │ You: What's the weather?            │
    │                                     │
    │ 🤔 Thinking...                      │
    │    (forever)                        │
    └─────────────────────────────────────┘
    
    Solution:
    1. Check backend window is open
    2. Look for errors in backend window
    3. Check if Ollama is running: ollama list
    4. Restart with START_NEXUS_AI.bat


STOPPING NEXUS AI
═══════════════════════════════════════════════════════════════════════

Option 1: Close Backend Window
────────────────────────────────────────────────────────────────────
    1. Find the backend window (green/black text)
    2. Click the [X] button to close
    3. Or press Ctrl+C inside the window
    4. Done! Nexus AI is stopped


Option 2: Kill Process
────────────────────────────────────────────────────────────────────
    Open PowerShell and run:
    
    Get-Process python | Stop-Process -Force


RESTARTING NEXUS AI
═══════════════════════════════════════════════════════════════════════

    1. Close backend window (if open)
    2. Double-click START_NEXUS_AI.bat
    3. Wait 10 seconds
    4. Browser opens (or refresh existing one)
    5. Start asking questions again!
